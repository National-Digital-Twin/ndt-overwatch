# NDTP Suppliers Register

**Repository:** `.github`
**Description:** `Provides a consolidated, de-duplicated list of organisations acknowledged across public repositories within the National Digital Twin Programme GitHub organisation.`
**SPDX-License-Identifier:** `OGL-UK-3.0`

The National Digital Twin Programme (NDTP) maintains this document as a consolidated register of organisations that have been acknowledged for contributing to publicly available repositories within the NDTP GitHub organisation.

## Purpose

This register provides a single, alphabetically ordered list of supplier organisations that have been formally recognised within individual repository `ACKNOWLEDGEMENTS.md` files.

It is intended to:

- Improve transparency of supplier engagement across the portfolio
- Provide a consistent reference point for reporting and governance purposes
- Reduce duplication across repositories

## Scope

This register:

- Includes only organisations listed under the **“Organisational contributions”** section of `ACKNOWLEDGEMENTS.md` files
- Covers **public repositories only** within the NDTP GitHub organisation
- Does not include individual contributors
- Does not imply contractual status, endorsement, or ongoing engagement

Private repositories are explicitly excluded from this aggregation.

## Method of Derivation

The list is generated through an automated process that:

1. Enumerates all public repositories within the NDTP GitHub organisation
2. Identifies the `ACKNOWLEDGEMENTS.md` file within each repository (where present)
3. Extracts organisation names listed under the `## Organisational contributions` heading
4. Normalises formatting and removes duplicates using case-insensitive comparison
5. Produces a single alphabetically ordered register

The process is deterministic and reproducible.

Updates to this register reflect the current state of acknowledged organisational contributions at the time of generation.
